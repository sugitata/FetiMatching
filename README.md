## Description
フェチが同じ人同士をマッチングさせる恋愛系サービスです
こちらは2016年夏にweb系スクールの卒業制作物として作成しました

## Environment
- Ruby 2.3.3
- Ruby on Rails 4.2.4

## How to start
```
$ git clone <This repository>
$ cd <This repository>
$ rbenv exec bundle install
$ rbenv exec bundle exec rails s -b 0.0.0.0
```
